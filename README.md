## 四大数据增广方法分别做了以下操作：
1. 水平翻转/垂直翻转：将图像水平或垂直翻转，即左右对称或上下对称。这有助于解决平移不变性问题。
2. 随机旋转：随机旋转图像一定的角度。这有助于解决旋转不变性问题。
3. 随机裁切：在随机位置截取并缩放图像。这有助于解决尺寸不变性问题。
4. 随机色度变换：随机调整图像的亮度、饱和度和对比度等颜色属性。这有助于解决光照复杂性问题。
